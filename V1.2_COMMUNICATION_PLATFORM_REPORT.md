# WILMS v1.2.0 — Communication Platform Completion Report

**Release:** v1.2.0  
**Branch:** `feature/v1.2.0-communication-platform`  
**Date:** 2026-07-07

## Executive summary

v1.2.0 completes the Communication Platform roadmap on top of v1.1.3. The release delivers a rich text composer, attachment system, password reset UI, push notification infrastructure, email open/click tracking, provider webhooks, expanded analytics, recurring scheduler, template builder with variables, and user notification preferences.

## Architecture

- **Migration:** `0016_v120_communication_platform.sql`
- **Tracking:** `dispatchMail()` injects pixel + click URLs → frontend `/api/t/*` → backend `/tracking/*`
- **Webhooks:** `/webhooks/mail/resend`, `/webhooks/mail/generic`
- **Attachments:** `message_attachments` + existing upload infrastructure
- **Preferences:** `user_notification_preferences` with channel/category gates in `sendMessage()`

## Key files

- Backend: `communication-platform.ts`, `email-tracking.ts`, `attachments.service.ts`, `password-reset.service.ts`
- Frontend: `RichTextEditor.tsx`, `AttachmentUploader.tsx`, `CommunicationCenterPanel.tsx`, `TemplateBuilderModal.tsx`
- Auth: `/forgot-password`, `/reset-password`

## Regression note

All v1.1.3 Communication Center routes and notification flows remain intact. No existing APIs were removed.
