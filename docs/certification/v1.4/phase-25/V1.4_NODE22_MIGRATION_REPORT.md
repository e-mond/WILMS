## `verify:node` implementation

Script: `scripts/verify-node-version.mjs`

Validates four invariants in one pass:

1. Current process major ≥ 22
2. Root `package.json` `engines.node` references 22
3. `.nvmrc` is `22` (or starts with `22`)
4. `Dockerfile` matches `/FROM node:22\b/`

Exit code 1 on any failure; used in CI `validate` job.
