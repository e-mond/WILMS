# P14.3B Phase 3C.2 — Reversal Certification

**Date:** 2026-06-21

---

## Summary

Production validation and certification for the Phase 3C.1 payment reversal MVP. Adds certification harness scripts and evidence reports; **no new reversal types or API changes**.

## Certification tooling

```bash
npm run cert:reversal:seed-reset -w @wilms/api   # Neutralize harness seed pollution
npm run cert:reversal:env -w @wilms/api
npm run cert:reversal:functional -w @wilms/api
npm run cert:reversal:ledger -w @wilms/api
npm run cert:reversal:concurrency -w @wilms/api
npm run cert:reversal:perf -w @wilms/api          # Optional: tsx ... 500 1000
```

- Additional reversal types remain blocked pending approval (3C.1 STOP gate).
