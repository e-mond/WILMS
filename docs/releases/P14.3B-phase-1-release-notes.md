## Features

- **Loan pool persistence** — `loan_pools`, `pool_memberships`, `pool_allocations` tables (migration `0003_loan_pools`)
- **Read API** — `GET /loan-pools`, `GET /loan-pools/:id` aligned with frontend `ILoanPoolService`
- **Allocation model** — Append-only pool movements (`DISBURSEMENT`, `REPAYMENT`, `REPLENISHMENT`, `ADJUSTMENT`)
- **Seed data** — 3 pools linked to financial seed loans via `loans.loan_pool_id`
- **Verification** — `npm run verify:pools -w @wilms/api` (5/5 checks)
