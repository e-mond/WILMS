# P14.3B Phase 3C.1 — Payment Reversal MVP

**Date:** 2026-06-20

---

## Summary

Implements **payment reversal only** for P14.3B Phase 3C.1:

- Migration `0005_financial_reversals.sql`
- `POST /payments/:paymentId/reverse` (RBAC: `ACCESS_ADMIN_PORTAL`)
- Compensating `REVERSAL` ledger entries
- `financial_reversals` + `reversal_history` audit trail
- Idempotency scope `REVERSAL_EXECUTE`

## Not included

- Disbursement reversal
- Pool allocation reversal
- Adjustment reversal
- Write-off reversal

## Verification

```bash
npm run db:migrate -w @wilms/api
npm run verify:reversals -w @wilms/api
```

## API

```http
POST /api/v1/payments/{paymentId}/reverse
Authorization: Bearer <session>
Idempotency-Key: <optional>

{
  "reason": "Duplicate collection posted in error",
  "actorId": "<user-id>",
  "actorDisplayName": "Super Admin"
}
```

Response: reversal result with before/after balance snapshot.
