## Railway Deploy

```bash

railway up
```

**Verify:**

```bash
curl -s https://wilms-production.up.railway.app/health | jq '.data.version, .data.migrations'
# Expected: "0.2.1", applied=7, status=ok
```

## Vercel Deploy

```bash
vercel deploy --prod
```

**Verify:**

```bash
curl -sI https://wilms.vercel.app/login | head -1
# Expected: HTTP/2 200
```

**Env vars (unchanged):**

- `NEXT_PUBLIC_API_BASE_URL` → Railway API
- `WILMS_API_UPSTREAM` → Railway API
- `NEXT_PUBLIC_USE_MOCK=false`
