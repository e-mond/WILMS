**Secondary factors:**

1. **`WILMS_GIT_COMMIT` unset** — cannot verify SHA via `/health` even after correct deploy
2. **11:11 redeploy** — likely redeploy of **previous successful image** (`a1e5f67` era), not fresh build from `main` @ `f69427c`

**Ruled out (this incident):**

- Migration failure on 0007 — active container only has 7 migrations in journal; 0007 never applied because v0.2.2 code never ran
- Webhook never received — webhook received; deploy was **SKIPPED**
