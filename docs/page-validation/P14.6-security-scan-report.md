## npm audit Findings (2026-06-30)

**Summary:** 22 vulnerabilities (2 critical, 9 high, 11 moderate)

### Notable — exceljs / uuid chain

| Package | Severity | Advisory | Fix |
|---------|----------|----------|-----|
| `uuid` | high | GHSA-w5hq-g745-h8pq — buffer bounds in v3/v5/v6 | Transitive via exceljs |
| `exceljs` | — | Depends on vulnerable `uuid` | `npm audit fix` → exceljs@3.4.0 (breaking) |

**Risk acceptance:** `exceljs` used for server-side report export only; not exposed to untrusted UUID input from clients. Upgrade tracked for v0.2.3.

### Other high findings

| Package | Severity | Context |
