# Export & Print Validation

Recorded: 2026-06-09

---

## Official document standard

### PDF exports (`pdf-engine.ts`)

| Requirement | Status | Implementation |
|---|---|---|
| WILMS branding | Pass | `drawDocumentHeader()` — green header bar, org name |
| Logo / wordmark | Pass | Text wordmark `WILMS` + full org name |
| Report title | Pass | `metadata.reportTitle` in header |
| Report ID | Pass | `generateReportId()` — e.g. `WILMS-BOR-2026-000001` |
| Generated date | Pass | `formatExportTimestamp()` in metadata |
| Generated by | Pass | `useWilmsExportActor()` / session user |
| Page numbers | Pass | `drawFooters()` — `Page N of M` on every page |
| Audit metadata | Pass | Audit Information section in tabular builder; footer report ID |
| A4 compatibility | Pass | `format: 'a4'`, portrait/landscape from document |
| Multi-page support | Pass | `doc.addPage()` when cursor exceeds page height |
| Table clipping | Pass | `jspdf-autotable` with `overflow: 'linebreak'` |

### Print exports (`print-engine.ts`)

| Requirement | Status | Implementation |
|---|---|---|
| WILMS branding | Pass | `.wilms-header` green block with wordmark |
| Logo | Pass | Text wordmark (matches PDF) |
| Report title | Pass | Header meta `h1` |
| Report ID | Pass | Header + footer |
| Generated date | Pass | Header meta + footer |
| Generated by | Pass | `.wilms-meta-bar` |
| Page numbers | Pass (P11d) | `@page @bottom-center { content: "Page " counter(page) }` |
| Audit metadata | Pass | Executive summary + Audit Information section |
| A4 margins | Pass | `@page margin: 14mm` |
| No shell chrome | Pass | Dedicated iframe HTML — no sidebar/nav |
| Table breaks | Pass | `.wilms-section { break-inside: avoid }` |
| Multi-page | Pass | Browser print pagination |

**Note:** Print page totals (`Page N of M`) depend on browser Paged Media support. PDF always includes full page counts via jsPDF.

## Registration documents

### On-screen / print preview (`RegistrationAgreementDocument.tsx`)

| Requirement | Status |
|---|---|
| 2-page layout | Pass — `.registration-agreement-page` with `print:break-after-page`, `min-h-[277mm]` |

| Applicant photo | Pass — `Avatar` / image in applicant section |

| Thumbprint placeholders | Pass — thumbprint area + "applied on printed copy" note |
| Legal notice | Pass — dedicated section from `legal.legalNotice` |
| Officer verification | Pass — officer signature block + registration metadata |

### Registration PDF/print export (`registration-agreement-document.ts`)

| Requirement | Status | Notes |
|---|---|---|
