## Permission Model (code truth)

`APPROVER` permissions (`packages/shared-rbac/src/role-permissions.ts`):

- `access-approver-portal`, `review-applications`, `approve-loans`, `reject-loans`, `approve-borrowers`, `review-risk-flags`

**Does not include** `view-financial-reports`. Therefore:

- ✅ Approver **can** create / approve / disburse loans
- ❌ Approver **cannot** list all loans, view schedule, or progress (requires super admin or auditor-equivalent reporting permission on routes)

Route guards: `apps/backend/src/modules/loans/routes.ts` uses `requirePermission(PERMISSION.VIEW_FINANCIAL_REPORTS)` for read endpoints.
