# P14.5D — Production Environment Matrix

**Date:** 2026-06-29  
**Important:** The P14.5D brief uses some variable names that **do not exist** in the WILMS codebase. This matrix lists **actual** names from `env.ts`, `validate-env.ts`, and frontend config.

---

## Brief name → actual name mapping

| Brief (P14.5D) | **Actual WILMS variable** | Platform |
|----------------|---------------------------|----------|
| `SESSION_SECRET` | **`WILMS_SESSION_SECRET`** | Railway |
| `NEXT_PUBLIC_API_URL` | **`NEXT_PUBLIC_API_BASE_URL`** (BFF path) + **`WILMS_API_UPSTREAM`** (server) | Vercel |
| `NEXTAUTH_URL` | **Not used** — custom HMAC session via `/api/auth/login` | — |
| `NEXTAUTH_SECRET` | **Not used** — use `WILMS_SESSION_SECRET` on backend only | — |
| `CLOUDINARY_URL` | **Not used** — use `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET` | Railway |
| `FRONTEND_URL` | **`WILMS_CORS_ORIGIN`** (backend) | Railway |

## Localhost risk (production must not use)

| Location | Fallback | Mitigation |
|----------|----------|------------|
| BFF `route.ts` | `127.0.0.1:4000` dev only | Throws if `WILMS_API_UPSTREAM` missing in production |
| `env.ts` `corsOrigin` | `http://127.0.0.1:3000` | `validate-env.ts` requires explicit `WILMS_CORS_ORIGIN` in production |
| Smoke script | — | Fails if URLs contain localhost or `http://` |
