## Production Guard

**Backend** — `assertProductionMockDisabled()` in `apps/backend/src/config/mock-guard.ts`:

| Flag | Action if `true` in production |
|------|------------------------------|
| `NEXT_PUBLIC_USE_MOCK` | `process.exit(1)` |
| `NEXT_PUBLIC_DEMO_MODE` | `process.exit(1)` |
| `NEXT_PUBLIC_FORCE_DEMO_MODE` | `process.exit(1)` |
| `NEXT_PUBLIC_API_DISABLED` | `process.exit(1)` |

Called at API startup (`apps/backend/src/index.ts`) before server bind.

**Frontend** — `services/index.production.ts` aliases mock services away at build time (`next.config.mjs` webpack alias). CI builds with `NEXT_PUBLIC_USE_MOCK: 'false'`.

**Deploy pipeline** — `deploy-production.yml` verification includes mock-flag check via smoke tests; hard-fail if localhost detected in production network calls.
