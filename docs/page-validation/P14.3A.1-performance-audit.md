* Sort ascending; p95 = ceil(0.95 × n)th sample

## Code-Path Analysis (Evidence-Based)

|-----------|---------------|----------|
| `listLoans` | Single SELECT loans | None |
| `getLoanSchedule` | SELECT schedules by loan_id | None |
| `listPortfolioEntries` | SELECT loans + N× borrower lookup | **Yes** — one borrower query per loan |
| `postPayment` | Single transaction (~5 writes) | None |
| `disburseLoan` | Single transaction (~4 writes) | None |
