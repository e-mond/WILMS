## Railway (API) — set in Railway dashboard

| Variable | Required | Notes |
|----------|----------|-------|
| `DATABASE_URL` | **Yes** | Neon PostgreSQL connection string |
| `WILMS_SESSION_SECRET` | **Yes** | ≥32 random chars (64 hex recommended) |
| `WILMS_CORS_ORIGIN` | **Yes** | `https://wilms.vercel.app` |
| `UPLOAD_PROVIDER` | **Yes** | `cloudinary` in production |
| `CLOUDINARY_CLOUD_NAME` | **Yes** | Cloudinary dashboard |
| `CLOUDINARY_API_KEY` | **Yes** | Cloudinary dashboard |
| `CLOUDINARY_API_SECRET` | **Yes** | Cloudinary dashboard — **secret** |
| `CLOUDINARY_FOLDER` | No | e.g. `wilms-prod` |
| `NODE_ENV` | Auto | `production` in container |
| `WILMS_TRUST_PROXY` | **Yes** | `true` |
| `WILMS_TRUST_PROXY_HOPS` | No | `1` |
| `WILMS_GIT_COMMIT` | Optional | Set at deploy for `/health` |
| `WILMS_API_HOST` | No | `0.0.0.0` (Dockerfile default) |
| `WILMS_API_PORT` | No | Railway injects `PORT` |

## Local development (`.env` / `.env.local`)

Copy from `.env.example` and `apps/frontend/.env.local.example`:

| Variable | Local typical value |
|----------|---------------------|
| `DATABASE_URL` | Neon dev branch or empty (in-memory API) |
| `WILMS_SESSION_SECRET` | Any 32+ char string for local |
| `WILMS_API_UPSTREAM` | `http://127.0.0.1:4000` |
| `NEXT_PUBLIC_API_BASE_URL` | `http://127.0.0.1:3000/api/wilms` |
| `NEXT_PUBLIC_USE_MOCK` | unset (mock default) or `false` for live API |

## Where to obtain real secrets

| Credential | Source |
|------------|--------|
| Neon `DATABASE_URL` | [Neon Console](https://console.neon.tech) → project → connection string |
| Cloudinary | [Cloudinary Console](https://cloudinary.com/console) |
| `WILMS_SESSION_SECRET` | Generate: `node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"` |
| Railway token | Railway → Account → Tokens |
| Vercel token | Vercel → Settings → Tokens |

---

## Files that must never be committed

- `.env`, `.env.local`, `.env.production`
- `.wilms-production-credentials.json`
- Any file containing live API keys or DB passwords
