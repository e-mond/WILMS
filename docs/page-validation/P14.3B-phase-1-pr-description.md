# PR: [P14.3B-Phase-1] Loan Pools — Read API & Persistence

**Base:** `release/p14.3a`

**Compare:** https://github.com/e-mond/WILMS/compare/release/p14.3a...feature/p14.3b-phase-1?expand=1

## Summary

Implements P14.3B Phase 1 loan pool persistence and read-only API aligned with the frontend `ILoanPoolService` contract. Includes migration `0003_loan_pools`, seed data, `verify:pools` harness (5/5), and Phase 1 certification documentation.

## Scope

**In scope**

- Schema: `loan_pools`, `pool_memberships`, `pool_allocations`
- `GET /loan-pools`, `GET /loan-pools/:id`
- RBAC: `VIEW_FINANCIAL_REPORTS`
- Seed + verification harness

**Out of scope (deferred)**

- Pool write HTTP API
- Disburse/payment pool allocation hooks
- Runtime audit writes for pool operations
