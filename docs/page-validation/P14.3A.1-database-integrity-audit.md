## Automated Integrity Checks

Implemented in `runDatabaseIntegrityChecks()`:

| Check | Validates |
|-------|-----------|
| `no-schedules-without-loans` | Schedule rows reference existing loans |
| `no-orphan-ledger-payment-refs` | REPAYMENT ledger `payment_id` exists in payments |
| `no-duplicate-payment-tuples` | No duplicate (borrower, date, amount) |
| `no-duplicate-disbursements-per-loan` | At most one disbursement row per loan |
| `no-negative-loan-balances` | `loan_balance >= 0` |

Unique idempotency: index `idempotency_scope_actor_key_idx` on `(scope, actor_user_id, idempotency_key)` — `db/schema/idempotency-keys.ts`.
