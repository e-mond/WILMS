## Scope

---

## Changes Implemented

| Area | Change | Files |
|------|--------|-------|
| CSRF | Synchronizer token on BFF mutations (`wilms_csrf` cookie + `x-wilms-csrf` header) | `apps/frontend/src/lib/auth/csrf*.ts`, BFF auth/wilms routes |
| Health | Expanded `/health`: version, git commit, uptime, runtime (node, deployedAt, buildId), migration count | `apps/backend/src/modules/health/health.service.ts` |
| Headers | Helmet: `hidePoweredBy`, explicit HSTS, `referrerPolicy: no-referrer` | `apps/backend/src/http/app.ts` |
| Fingerprints | Next.js `poweredByHeader: false` (from P14.5F) | `apps/frontend/next.config.mjs` |
| Version | `WILMS v{version}` from root `package.json` on login, sidebar, dashboard footer, settings | `app-version.ts`, `AppVersionBadge`, `OfficeShellFooter`, `AppSidebar` |
| Demo users | Rotation script (suspend `@wilms.demo`, create `@wilms.production`) | `apps/backend/scripts/rotate-production-users.mjs` |
| Smoke tests | CSRF issuance, CSRF rejection, expanded health checks | `production-smoke.ts` |
