# P13.1 Photo Gap Closure

**Date:** 2026-06-15  
**Scope:** Close remaining P13 photo rendering gaps using existing `photoUrl` DTO fields and `resolveEntityPhotoUrl()`.

## Problem

P13 audit identified four surfaces still calling `resolvePersonPhotoUrl()` without passing entity `photoUrl`, causing Dicebear fallback even when mock/API data included uploaded photo URLs.

## Fixes Applied

| Surface | File | Change |
| ------- | ---- | ------ |
| Borrower profile (header + aside) | `src/features/borrower-management/components/BorrowerProfilePanel.tsx` | `resolveEntityPhotoUrl({ name, id, photoUrl: borrower.photoUrl })` |
| Group leader + assigned collector | `src/features/group-management/components/profile/GroupDetailSections.tsx` | Pass `leader.photoUrl` / `collector.photoUrl` |
| Collector list, aside, profile, mobile cards | `CollectorsManagementPanel.tsx`, `CollectorsAsidePanel.tsx`, `CollectorProfilePanel.tsx`, `CollectorsMobileCardList.tsx` | Pass `row.photoUrl` / `data.photoUrl` |
| Settings users list + profile modal | `SettingsUsersSection.tsx`, `SettingsUserProfileModal.tsx` | Pass `row.photoUrl` / `data.profileImageUrl` |

## Mock / Type Support

| Type | Field added/used | Populated in |
| ---- | ---------------- | ------------ |
| `GroupLeaderProfile` | `photoUrl` | `buildGroupLeaderProfile()` from member `photoUrl` |
| `GroupCollectorAssignment` | `photoUrl` | `buildCollectorAssignment()` from collector demo dataset |
| `SettingsUserProfile` | `profileImageUrl` | `buildUserProfileFromRecord()` from `user.photoUrl` |

## Resolution Order (unchanged)

```ts
resolveEntityPhotoUrl(input) → resolvePersonPhotoUrl({ ..., uploadUrl: input.photoUrl })
```

Uploaded URL wins; Dicebear only when `photoUrl` is absent.

## Validation

- `npm run type-check` — pass
- `npm run test` — 400 passed (200 + 200 shards)

## Intentionally Unchanged

- `RoleSettingsPanel` profile photo already passes `uploadUrl` from `PhotoUpload` callback (working).
- Surfaces fixed in P13 (borrower list, search, notifications, audit log, group members, approval review) were not modified.
