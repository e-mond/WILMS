## Reconciliation Rules

For each seeded loan (`seed-financial.ts` IDs `01930002-…`):

```text
Loan Principal = Disbursement Total        (when disbursed)
Repayment Ledger Total = Payment Table Total
loan_balance = principal − sum(payments)
Schedule installment sum = principal
PAID schedule weeks = payment row count
No orphan REPAYMENT ledger rows
balance ≥ 0
```

---

## Verification Implementation

`apps/backend/src/verification/db-checks.ts` → `runLedgerConsistencyChecks()`

Per-loan checks emitted:

| Check name | Validates |
|------------|-----------|
| `disbursement-equals-principal-{id}` | LOAN_DISBURSEMENT ledger + disbursement table |
| `repayment-ledger-matches-payments-{id}` | REPAYMENT ledger sum = payments.amount_pesewas sum |
| `balance-reconciles-{id}` | `loan_balance` vs computed remainder |
| `no-negative-balance-{id}` | balance ≥ 0 |
| `schedule-sum-equals-principal-{id}` | schedule installments sum |
| `paid-weeks-match-payments-{id}` | PAID weeks vs payment count |
| `no-orphan-repayment-ledger-{id}` | every REPAYMENT ledger row has matching payment FK |

## Portfolio Reconciliation

Implemented in `runPortfolioReconciliationChecks()`:

```text
Total Disbursed − Total Repaid = Expected Outstanding
Expected Outstanding = Sum(ACTIVE + DEFAULTED loan balances)
COMPLETED loans → balance = 0
```

Checks: `portfolio-disbursements-minus-repayments`, `completed-zero-balance-{id}`.
