| ------- | -------- | ------------------- |

## Changes Made (no arbitrary timeout increases)

- Removed duplicate overlay hydration that caused strict locator collisions.
- Replaced ambiguous locators (`table`, `Open Flags`, `dialog` without name) with scoped selectors.
- Added deterministic sign-in option for app-lock test without disabling global mandatory gate for other specs.
- Operational mobile nav helper waits for visible `role="tab"` instead of absent drawer button.
