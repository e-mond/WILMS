# P13.1 Signature & Thumbprint Upload Integration

**Date:** 2026-06-15

## Previous State

Registration step 5 stored signature/thumbprint captures as base64 data URLs in form fields (`borrowerSignature`, etc.). `SignaturePad` and `IdentityCaptureField` persisted canvas output directly in react-hook-form state.

## Implementation

### Upload purposes

Added to `src/types/upload.ts`:

- `UPLOAD_PURPOSE.SIGNATURE` (`signature`)
- `UPLOAD_PURPOSE.THUMBPRINT` (`thumbprint`)

### Utilities (`src/utils/upload-file.ts`)

- `dataUrlToFile()` — canvas → `File` for service upload
- `uploadDataUrlViaService()` — upload drawn capture via `IUploadService`
- `resolveUploadPreviewUrl()` — preview/delete/replace via stored upload ID

### Components

| Component | Role |
| --------- | ---- |
| `IdentityCaptureField.tsx` | Borrower/guarantor signature + thumbprint; upload/replace/preview/delete via service |
| `SignatureUploadField.tsx` | Officer signature pad with same upload lifecycle |

### Form state (upload IDs only)

Replaced base64 string fields with:

- `borrowerSignatureUploadId`, `borrowerThumbprintUploadId`
- `guarantorSignatureUploadId`, `guarantorThumbprintUploadId`
- `officerSignatureUploadId`
- `borrowerThumbprintManualPlaceholder`, `guarantorThumbprintManualPlaceholder` (boolean flags for print placeholder UX)

Updated: `registration.schema.ts`, `registration.utils.ts`, `types/borrower-registration.ts`, `RegisterBorrowerPayload`.

### Review / print / PDF

- `src/utils/registration-media.ts` — resolves upload IDs to URLs for agreement content
- `RegistrationReviewPanel.tsx` — async media resolution before preview/export
- `registration-agreement-fields.ts` — accepts resolved URLs in `RegistrationAgreementMedia`
- `registration-agreement-document.ts` — export uses resolved media (no base64 in form)

## Operations Matrix

| Area | Upload | Preview | Replace | Delete |
| ---- | ------ | ------- | ------- | ------ |
| Borrower signature | ✅ `IdentityCaptureField` | ✅ from `getUpload` | ✅ retake + re-upload | ✅ `deleteUpload` on retake |
| Borrower thumbprint | ✅ draw or file | ✅ | ✅ | ✅ |
| Guarantor signature/thumbprint | ✅ | ✅ | ✅ | ✅ |
| Officer signature | ✅ `SignatureUploadField` | ✅ | ✅ | ✅ |
| Registration review / PDF | ✅ resolved URLs | ✅ `RegistrationAgreementDocument` | N/A (read-only) | N/A |
