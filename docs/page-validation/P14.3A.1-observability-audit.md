## Audit Trail

**In-memory:** `infrastructure/audit/audit-log.ts` — synchronous append to array.

**Database:** `repositories/audit.repository.ts` — async fire-and-forget (`void …catch(console.error)`).

Loan actions mapped to enum values (post `0002_sad_bucky.sql`):

* `loan.created` → `LOAN_CREATED`
* `loan.approved` → `LOAN_APPROVED`
* `loan.rejected` → `LOAN_REJECTED`
* `loan.disbursed` → `LOAN_DISBURSED`
* `payment.recorded` → `PAYMENT_RECORDED`

Target entity `loan` → `LOAN` enum.

**Gap:** Failed DB audit writes are logged to stderr only — no alert/metric.
