## Data-fetching patterns

### TanStack Query (`isLoading` / `isPending`)

Dominant pattern across feature panels. Representative files (60+ matches for `isLoading`/`LoadingSpinner`):

| Domain | File | Pattern |
|--------|------|---------|
| Reports | `DailyCollectionReportPanel.tsx`, `ReportsIndexPanel.tsx`, `LoanPortfolioReportPanel.tsx`, … | `useQuery` + conditional `<LoadingSpinner />` |
| Settings | `SettingsPanel.tsx`, `SettingsUsersSection.tsx`, `SettingsAsidePanel.tsx` | Query per section |
| Dashboard | `SuperAdminDashboard.tsx`, `DashboardCollectionSummary.tsx`, `DashboardExpenseSummary.tsx` | Parallel queries |
| Borrowers / loans | `BorrowerList.tsx`, `LoanPortfolioList.tsx`, `CreateLoanWizard.tsx` | List + detail loaders |
| Collectors | `CollectorsManagementPanel.tsx`, `CollectorProfilePanel.tsx` | Management + profile |
| Risk flags | `RiskFlagsPanel.tsx` | Table load state |
| Payments | `PaymentEntryPanel.tsx`, `CollectorDashboardPanel.tsx` | Collection workflows |
| Auth / permissions | `PermissionProvider.tsx`, `PermissionRouteGuard.tsx`, `usePermissions.ts` | Gate until session resolved |

### Permission gating

`PermissionRouteGuard.tsx` and `PermissionProvider.tsx` show loading until auth + permissions hydrate — prevents flash of unauthorized content (ADR-002).
