## Production Integration Path (documented, not verified live)

1. Set `DATABASE_URL`, fix seed, run migrate + seed
2. Run backend with Neon: `[postgresql (Neon)]` in startup log
3. Set `NEXT_PUBLIC_API_BASE_URL` to BFF or backend URL
4. Set `WILMS_API_UPSTREAM=http://127.0.0.1:4000`
5. Run frontend with `NODE_ENV=production` build/start
