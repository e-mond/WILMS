## Loan Calculations

| Check | Rule | Result | Evidence |
|-------|------|--------|----------|
| Weekly installment | `Math.floor(amount / weeks)` | Ô£à 50000├À10 = 5000 | `calculations.ts` |
| Divisibility gate | `amount % weeks === 0` | Ô£à Enforced at create | `assertDivisibleLoanAmount()` |

| Outstanding balance | Decremented per repayment | Ô£à `loanBalance` update | `postPayment()` |

### Remaining Principal / Interest

P14.3A scope: zero-interest loans. `interest_amount` and `penalty_amount` default to `0`. Remaining principal equals `loan_balance`. No separate interest accrual engine ÔÇö **Safe** per P14.3A design boundary.
