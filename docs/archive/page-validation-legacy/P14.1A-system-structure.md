## Overview

WILMS is a monorepo-style workspace:

| Package | Path | Runtime |
|---------|------|---------|
| Frontend (Next.js 14) | repository root (`src/`) | Port 3000 (default) |
| Backend API (Express) | `backend/` | Port 4000 (default) |

Frontend remains the source of truth for domain types and UI. Backend uses in-memory persistence (`backend/src/db/store.ts`). Neon + Drizzle migration has **not** started.

Provider selection: `src/data-provider/` switches mock vs API via `resolveDataProviderMode()` and webpack aliases (`index.development.ts` / `index.production.ts`).
