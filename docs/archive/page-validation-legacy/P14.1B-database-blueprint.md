# P14.1B ÔÇö Database Table Blueprint

**Phase:** P14.1B (blueprint only ÔÇö no SQL, no Drizzle)  
**Date:** 2026-06-09  
**Evidence:** Type interfaces, backend records, mock stores. Table names are blueprint labels mapped from record/type names in code.

**Not in code today:** Most entities lack `updatedAt`, `createdBy`, `updatedBy`, or soft-delete flags. Where absent, blueprint marks **Not in current types** rather than inventing columns.

---

## Blueprint conventions

| PostgreSQL type (blueprint) | TypeScript source pattern |
|----------------------------|---------------------------|
| `uuid` / `text` | `string` id fields |
| `text` | `string` |
| `integer` | `number` (counts, pesewas) |
| `boolean` | `boolean` |
| `timestamptz` | ISO `string` dates |
| `jsonb` | Nested objects (e.g. GPS) |
| Enum | `as const` unions in types/constants |
