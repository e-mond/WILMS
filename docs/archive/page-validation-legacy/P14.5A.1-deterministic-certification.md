# P14.5A.1 ÔÇö Deterministic Certification

**Goal:** Repeated certification runs produce identical pass/fail outcomes on shared Neon.

---

## Determinism mechanisms

### Financial certification (`verify:financial`)

| Step | Mechanism | File |
|------|-----------|------|
| Pre-run reset | Delete seed-loan FK chain ÔåÆ re-seed financial core | `cert-financial-prep.ts` |
| Idempotency cleanup | Delete harness keys matching `idempotency-audit-%`, `concurrency-test-%`, `p143a4-%`, `cert-%` | `cert-financial-prep.ts` |
| Scoped assertions | Portfolio + integrity checks limited to `SEED_LOAN_IDS` | `db-checks.ts` |
| Unique concurrent keys | `concurrency-test-{N}-{uuid}-{index}` per run | `db-checks.ts` |
| Performance warmup | One call before sampling; Neon-specific budgets | `db-checks.ts` |
| Import safety | Prep `main()` only runs when script invoked directly | `cert-financial-prep.ts` |

### Live certification (`verify:live`)

| Step | Mechanism | File |
|------|-----------|------|
| Cert borrower reset | Upsert approved borrower with `hasActiveLoan=false` | `cert-live-prep.ts` |
| Harness loan cleanup | Delete all loans for `CERT_LIVE_BORROWER_ID` + FK chain | `cert-live-prep.ts` |
| Idempotency cleanup | Delete `p143a4-*` keys | `cert-live-prep.ts` |
| Target selection | Prefer cert borrower when loan-eligible | `live-integration.ts` |
| Import safety | Prep `main()` guarded | `cert-live-prep.ts` |

## Environment overrides

For CI or dedicated cert DB with local Postgres, defaults apply (300ms schedule, 250ms loan list):

```env
# Optional ÔÇö force stricter or looser budgets
WILMS_VERIFY_SCHEDULE_P95_MS=2500
WILMS_VERIFY_LOAN_LIST_P95_MS=750
```

Neon URLs (`neon.tech`, `neon.`) auto-select remote budgets without overrides.

| Script | Role |
|--------|------|

## Known non-deterministic elements (acceptable)

| Element | Mitigation |
|---------|------------|
| Neon query latency | Environment-aware p95 budgets + warmup |
| Absolute perf ms values | Assert against budget, not fixed ms |
| Cloudinary smoke | SKIP when creds absent ÔÇö infrastructure, not app |
