## Deploy Commands

### Railway API (monorepo root)

```bash
cd WILMS
railway link --project wilms   # service: WILMS
railway up
```

Uses `Dockerfile` + `railway.toml` at repo root. **Do not** deploy from `apps/backend/`.

### Vercel Frontend (monorepo root)

```bash
cd WILMS
vercel link --project wilms
vercel deploy --prod --yes
```

Uses root `vercel.json`:
- `installCommand`: `npm ci`
- `buildCommand`: `npm run build -w @wilms/frontend`
- `outputDirectory`: `apps/frontend/.next`

### Production smoke

```bash
WILMS_APP_URL=https://wilms.vercel.app \
WILMS_API_URL=https://wilms-production.up.railway.app \
npm run smoke:production
```
