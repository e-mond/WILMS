# P14.1C ÔÇö Table Architecture Blueprint

**Phase:** P14.1C (architecture only ÔÇö no SQL, no Drizzle)  
**Date:** 2026-06-09  
**Governing:** ADR-001, P14.1C mandatory rules  
**Evidence:** P14.1B-database-blueprint.md, ADR-001, phase brief

**Conventions (Approved):**

- Primary keys: UUID v7 (`id`)
- Timestamps: `created_at`, `updated_at` on mutable business tables
- Optimistic concurrency: `version` integer on mutable entities (phase brief)
- Soft delete: `deleted_at` timestamptz nullable where mandated
- Enums: PostgreSQL enum types sourced from `backend/src/contracts/enums/` (Proposed path ÔÇö ADR-001)
