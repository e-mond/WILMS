**Legend**

| Column | Meaning |
|--------|---------|
| **Frontend** | Feature UI + service interface + types exist |
| **Backend** | Express module/routes exist (may be stub/partial) |
| **Mock only** | Demo mode fully functional via `src/services/mock/*`; no matching backend |
| **API stub** | `src/services/*Service.ts` calls `apiClient` paths; backend missing or returns placeholder data |
| **Fully implemented** | Frontend mock **and** backend module both implement the domain flow with real logic (in-memory persistence counts) |

A domain can be **Mock only** and **API stub** simultaneously (typical P14.0 state: mock works, API client wired, backend incomplete).
