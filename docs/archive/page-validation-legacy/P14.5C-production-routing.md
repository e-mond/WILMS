## Routing model: BFF (Backend-for-Frontend)

WILMS uses **same-origin** API routing in production. The browser never calls the Railway API directly.

```text
Browser
  Ôöé
  Ôö£ÔöÇ GET /login, /loans, ÔÇª          ÔåÆ Vercel (Next.js pages)
  Ôöé
  Ôö£ÔöÇ POST /api/auth/login           ÔåÆ Vercel (Next route) ÔåÆ sets wilms_session cookie
  Ôöé
  ÔööÔöÇ GET/POST /api/wilms/*          ÔåÆ Vercel BFF ÔåÆ Railway Express
```

## URL configuration

| Role | Variable | Example |
|------|----------|---------|
| Public app | `NEXT_PUBLIC_APP_URL` | `https://app.wilms.example` |
| Client API base | `NEXT_PUBLIC_API_BASE_URL` | `https://app.wilms.example/api/wilms` |
| BFF upstream | `WILMS_API_UPSTREAM` | `https://wilms-api.up.railway.app` |
| CORS (backend) | `WILMS_CORS_ORIGIN` | `https://app.wilms.example` |

---

## Client routing (`apiClient.ts`)

```typescript
// apps/frontend/src/config/api.ts
export const API_BASE_URL = process.env.NEXT_PUBLIC_API_BASE_URL ?? '';
```

All service calls use relative BFF path ÔÇö **no hardcoded localhost** in production when env is set.

## BFF proxy (`route.ts`)

```typescript
// apps/frontend/src/app/api/wilms/[...path]/route.ts
// Production: WILMS_API_UPSTREAM required (throws if missing)
// Maps /api/wilms/loans ÔåÆ ${WILMS_API_UPSTREAM}/loans
```

Session cookie ÔåÆ `Authorization: Bearer` header for upstream.

---

## Backend route mounts

Express mounts business routes at **both** `/api/v1` and `/` (`app.ts` L65ÔÇô66). BFF forwards to upstream root paths (e.g. `/loans`, `/auth/login`).

## Localhost elimination checklist

| Check | Status |
|-------|--------|
| `NEXT_PUBLIC_API_BASE_URL` uses production domain | **Required at deploy** |
| `WILMS_API_UPSTREAM` uses Railway HTTPS URL | **Required at deploy** |
| `WILMS_CORS_ORIGIN` uses production domain | **Required at deploy** |
| No `127.0.0.1` in Vercel env | **Verify in dashboard** |
| BFF production guard | **Implemented P14.5C** |
| Backend `WILMS_API_HOST=0.0.0.0` | **Default in production** |

## Anti-patterns (do not use in production)

- `NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:4000` ÔÇö bypasses BFF
- `WILMS_API_UPSTREAM=http://localhost:4000` on Vercel ÔÇö unreachable
- `NEXT_PUBLIC_USE_MOCK=true` ÔÇö mock data in production
