## Upgrade strategy (ordered)

### Phase 1 — Safe patch (v0.2.3)

1. `npm audit fix` for dompurify-only fixes.
2. Re-run `npm run test`, `npm run build`, `npm run type-check`.

### Phase 2 — ORM security (v0.3.0)

1. Upgrade `drizzle-orm` to `0.45.2+` and matching `drizzle-kit`.
2. Full migration dry-run on staging (`STAGING_DATABASE_URL`).
3. Re-run `node scripts/p14-6-4-api-matrix.mjs` + smoke tests.

### Phase 3 — Toolchain (v0.3.0+)

1. Playwright bump + `npm run test:e2e:install`.
2. Evaluate Next.js / postcss major upgrade separately (bundle budget + perf scripts).

### Explicitly avoid

- `npm audit fix --force` on production branch without staged QA — forces Next 16, exceljs downgrade, breaking drizzle-kit.
